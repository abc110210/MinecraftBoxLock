# 适配 1.21.1 带Gui界面箱子锁插件
## 功能
支持上锁类型: 箱子,发射器，投掷器，合成器，熔炉，高炉，烟熏炉
- 箱子自动上锁
- Gui界面操作
- 密码箱
- 漏斗保护
- 独立/全局 权限
- 多状态切换
### 独立权限
- 可单独对这个箱子的权限操作
- 这个权限是对单个箱子的操作
- 权限被共享后是可以破坏该箱子的
### 全局权限
- 相当于将你上锁的箱子全部共享给玩家
- 共享后的玩家是可以破坏你放置的箱子
### 锁定切换
- 可以将单个箱子设置为公开/私有状态
- 公开状态其他玩家只能打开，不能破坏
### 漏斗传输
- 默认新放置的箱子都会关闭漏斗传输
- 可以在默认菜单切换，切换之后放置的箱子默认解锁
### 自动上锁
- 默认放置的箱子会自动上锁(私有)
- 可以在默认菜单切换，切换之后放置的箱子默认解锁(公开)
### 密码箱
- 设置密码箱之后，无论该玩家是否拥有这个箱子授权都需要密码才能打开
- 如果这个玩家没有授权，知道密码也是能够打开这个密码箱的
- 密码箱只支持 4~8 位 英文和数字 组合
## 如何使用
- 将文件放到服务器的Plugins文件夹里面
